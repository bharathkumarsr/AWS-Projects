# AWS CloudTrail

## Objective

Enable auditing and governance across AWS resources.

---

## Trail Details

| Parameter | Value |
|----------|---------|
| Name | enterprise-devops-audit |
| Type | Multi-region |

---

## Configuration

Enabled:

- Management events
- Read events
- Write events

Storage:

- Amazon S3

---

## Events Captured

- EC2 creation
- S3 uploads
- IAM modifications
- RDS operations

---

## Why CloudTrail?

- Auditing
- Compliance
- Security investigation
- Governance

---

## Key Learning

CloudTrail records every API call performed in AWS.
