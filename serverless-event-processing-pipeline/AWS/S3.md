# Amazon S3 (Simple Storage Service)

## Overview

Amazon S3 (Simple Storage Service) is a highly scalable, durable, and secure object storage service provided by AWS. In this project, S3 serves as the entry point of the serverless event-driven architecture by storing uploaded files and triggering the AWS Lambda function whenever a new object is created.

---

# Purpose in This Project

Amazon S3 is responsible for:

- Storing uploaded files.
- Generating Object Created events.
- Triggering the Lambda function automatically.
- Acting as the source of the event-driven workflow.

# Event Notification Configuration

The S3 bucket is configured with an Event Notification.

### Event Name

```
FileUploadTrigger
```

### Event Type

```
All Object Create Events
```

This means whenever a new object is uploaded into the bucket, Amazon S3 automatically generates an event.

# How It Works

### Step 1

A user uploads a file into the S3 bucket.

Example

```
employee-list.csv
```

---

### Step 2

Amazon S3 stores the object securely.

---

### Step 3

S3 automatically generates an **ObjectCreated** event.

Example Event

```json
{
  "eventName": "ObjectCreated:Put"
}
```

---

### Step 4

The event notification invokes the Lambda function.

```
ServerlessEventProcessor
```

---

### Step 5

Lambda receives:

- Bucket Name
- Object Key (File Name)
- Event Time
- Region
- Request Information

---

### Step 6

Lambda processes the uploaded file and performs:

- Reads object metadata
- Retrieves file size
- Retrieves content type
- Stores metadata into DynamoDB
- Sends an email notification using SNS
- Writes execution logs to CloudWatch

---

# Security Configuration

The bucket is secured using AWS best practices.

- Block Public Access enabled
- Private bucket
- IAM Role used for secure access
- Lambda accesses bucket using IAM permissions
- No public object access

---

# IAM Permissions Required

Lambda requires the following S3 permissions:

```json
{
  "Effect": "Allow",
  "Action": [
    "s3:GetObject",
    "s3:HeadObject"
  ],
  "Resource": "arn:aws:s3:::bharath-serverless-pipeline-luffy/*"
}
```
