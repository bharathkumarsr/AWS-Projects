# Amazon CloudWatch Monitoring

## Objective
Monitor the health and performance of the production infrastructure.

## Dashboard Name
prod-devops-dashboard

## Metrics Added

- EC2 CPU Utilization
- EC2 Network In
- EC2 Network Out
- Application Load Balancer Request Count
- Healthy Target Count

## Benefits

- Real-time monitoring
- Infrastructure performance analysis
- Capacity planning
- High availability monitoring
- Operational visibility

## AWS Services Used

- Amazon CloudWatch
- Amazon EC2
- Application Load Balancer
- Auto Scaling Group
