# 🚀 Production-Grade Highly Available AWS Infrastructure

## Project Overview

This project demonstrates the design and implementation of a production-grade AWS infrastructure following DevOps and cloud best practices.

The environment is designed to be secure, scalable, highly available, and deployment-ready.

---

## Architecture Components

- Amazon VPC
- Public & Private Subnets
- Internet Gateway
- NAT Gateway
- Route Tables
- Security Groups
- IAM Roles & Policies
- Amazon EC2
- Application Load Balancer (ALB)
- Auto Scaling Group
- AWS WAF
- Amazon Route 53
- AWS Certificate Manager (ACM)
- Amazon CloudWatch
- Amazon SNS
- Docker
- Jenkins
- GitHub Webhooks

---

## Project Objectives

- Design a secure AWS network architecture
- Deploy containerized applications
- Implement High Availability
- Configure CI/CD using Jenkins
- Secure applications with AWS WAF
- Configure DNS using Route 53
- Enable HTTPS using ACM
- Monitor infrastructure using CloudWatch
- Configure automated alerts using SNS
